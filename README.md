# 💙 Blog

A lightweight, responsive blog website built entirely with HTML, CSS, and vanilla JavaScript.
[Live Demo](https://blog-pi-coral-79.vercel.app/)

---

## 🚀 Quick Start

You don't need to install any dependencies or run a local server to view this project. It runs directly in your browser!
To get a local copy up and running, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/codedcollections/Blog.git
   ```
2. **Navigate into the project folder:**
   cd Blog

3. **Launch the app:**
   Double-click the index.html file to open it in a web browser.
